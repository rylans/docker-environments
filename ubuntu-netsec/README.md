# Ubuntu with network security tools

* Based off Ubuntu 17.10 (Artful)
* Comes with:
  * perl
  * python and pip
  * ping
  * vim
  * curl
  * wget
  * nmap
  * zmap
  * lynx
  * dnsutils (dig, nslookup, nsupdate)
  * net-tools (ifconfig, netstat, ipmaddr, arp, rarp)
  * inetutils-traceroute (traceroute)
  * less
  * ssh
  * man
  * Lynis 2.5.7
  * ivre
